# BasicViewer

グラフ描写アプリケーション。2D,3Dのグラフの描写とそのカスタマイズ、画像化などの機能。
微分方程式や数式文字列の数値計算結果のグラフやそのシミュレーションのアニメーション化などの機能も追加予定。
